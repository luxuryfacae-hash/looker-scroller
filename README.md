# Luxury Facilities Mobile App Prototype

This repository now contains a mobile web prototype (`scroller.html`) for the **Luxury Facilities** rebrand.

## What is included
- Rebranded color system inspired by the attached monogram icon (plum + cream).
- iOS-style mobile app frame.
- Home dashboard concept with:
  - Hero section
  - Core feature cards
  - Competitor comparison summary section
  - Bottom navigation

## Run locally
Open the file directly in a browser:

```bash
xdg-open scroller.html
```

Or serve with a static server:

```bash
python3 -m http.server 8080
```
Then visit <http://localhost:8080/scroller.html>.
